## Pendahuluan
PPDB Online adalah sistem pendaftaran siswa baru secara online untuk SMK KARYA BANGSA SINTANG.

### Persyaratan
- Komputer/smartphone dengan browser
- Koneksi internet
- Dokumen persyaratan (scan/foto)
- Bukti pembayaran

## Untuk Calon Siswa

### 1. Registrasi Akun
1. Kunjungi website PPDB
2. Klik tombol "Daftar"
3. Isi form registrasi
4. Verifikasi email

### 2. Login
1. Masukkan username
2. Masukkan password
3. Klik "Login"

### 3. Mengisi Formulir
1. Buka Dashboard
2. Isi data pribadi
3. Isi data orang tua
4. Pilih jurusan
5. Simpan formulir

### 4. Upload Dokumen
1. Siapkan scan/foto:
   - Ijazah
   - Kartu Keluarga
   - Akta Kelahiran
2. Upload masing-masing dokumen
3. Pastikan file tidak lebih dari 2MB

### 5. Pembayaran
1. Lihat nominal pembayaran
2. Transfer ke rekening sekolah
3. Upload bukti pembayaran
4. Tunggu verifikasi

### 6. Cek Status
1. Login ke dashboard
2. Lihat status pendaftaran
3. Tunggu pengumuman

## Untuk Admin

### 1. Login Admin
1. Akses halaman admin
2. Login dengan kredensial admin

### 2. Dashboard
- Lihat statistik pendaftaran
- Kelola data pendaftar
- Verifikasi dokumen
- Approve/reject pendaftaran

### 3. Verifikasi Pembayaran
1. Cek bukti pembayaran
2. Verifikasi nominal
3. Update status pembayaran

### 4. Manajemen Pendaftar
1. Lihat detail pendaftar
2. Download dokumen
3. Update status
4. Generate laporan

### 5. Pengaturan
- Kelola jurusan
- Atur kuota
- Kelola pengumuman
- Backup data

## Troubleshooting

### Masalah Umum
1. Lupa password
   - Gunakan fitur reset password
   - Hubungi admin

2. Gagal upload
   - Cek ukuran file
   - Cek format file
   - Coba kompres file

3. Status tidak update
   - Refresh halaman
   - Clear cache browser
   - Hubungi admin
